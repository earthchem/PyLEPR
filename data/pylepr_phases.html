    Phases in LEPR & traceDs

    Jump to bottom Edit New page
    Sarah Shi edited this page Apr 19, 2024 · 7 revisions

    Use the full name of the phase group in the "Description" column for chemical analyses.

    The species abbreviations in the "Abbreviation" column can be used a parameters for mineral modes present in experimental run products (or the starting material). Any of these abbreviations can be followed by "error" or "_err" to indicate uncertainties. Enter a value of "x" to denote a species that is "present not quantified".

    If a phase or species name / abbreviation does not appear on the list below, please add a new Issue.

    Notes
    • Chromite and Magnetite are listed with Spinel
    • Ferropericlase is synonymous with Magnesiowustite
    • The abbreviation "ak" is used here for Akimotoite; Akermanite is "aker"
    • The abbreviation "fb" is used here for Ferrobustamite; Ferropseudobrookite is "fpsb" or "armc"
    • The abbreviation "flu" is used here for Fluorite; Fluid is "fl"
    • The abbreviation "liq" is used here for silicate melts; Melilite is "mel"
    • Bridgmanite = (Mg,Fe)SiO3
    • CaSi perovskite = CaSiO3, a.k.a. Davemaoite
    • Perovskite = CaTiO3
    • Anhydrous B = (Mg,Fe)14Si5O24
    • Phase B = (Mg,Fe)12Si4O19(OH)2
    • Superhydrous B = (Mg,Fe)10Si3O14(OH)4
    • Phase F = (Mg,Fe)Si2O4(OH)2
